# GeoPlaces SPA Sample (ASP.NET Core + Angular Host)

This solution demonstrates:

- ASP.NET Core Web API (GeoPlaces.Web)
- PostgreSQL + PostGIS
- EF Core + Dapper
- Simple event contract + publisher
- xUnit tests
- Angular SPA hosted by ASP.NET Core (via `wwwroot`)
- Docker Compose for Postgres/PostGIS
